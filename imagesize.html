The Clown Car Technique test

Estelle Wey‘s solution, object tags that have svg data, the svg utilizes media queries and either a PNG or JPEG. Why this technique?

  • It doesn‘t rely on any other tools
  • It doesn‘st require the inclusion of a library
  • It doesn‘t rely on cookies
  • It doesn‘t send any unncessary requests
  • It accounts for all browsers and has a fallback for IE 8
The drawbacks:
  • viewBox is annoying and can‘t be easily manipulated
  • IE makes things more complicated
  • The code is kinda ugly
  • Doesn‘t easily work with percentage widths
